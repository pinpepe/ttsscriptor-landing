# Landing Page para TTS Scriptor

Este es el sitio web de marketing estático para el proyecto TTS Scriptor. Está construido con HTML, Tailwind CSS y JavaScript simple.

## Cómo Desplegar en Netlify

El despliegue es increíblemente sencillo y gratuito.

### Requisitos

*   Una cuenta de [GitHub](https://github.com/).
*   Una cuenta de [Netlify](https://netlify.com/).

### Pasos

1.  **Crear un Repositorio en GitHub:**
    *   Ve a tu cuenta de GitHub y crea un nuevo repositorio (por ejemplo, `ttsscriptor-landing`).
    *   Sube los archivos de este proyecto (`index.html`, la carpeta `assets/`, etc.) a tu nuevo repositorio.

2.  **Conectar Netlify a GitHub:**
    *   Inicia sesión en tu cuenta de Netlify.
    *   En tu dashboard, haz clic en el botón **"Add new site"** y luego selecciona **"Import an existing project"**.

3.  **Seleccionar el Repositorio:**
    *   Haz clic en el proveedor de Git **"GitHub"**.
    *   Busca y selecciona el repositorio que acabas de crear (`ttsscriptor-landing`).

4.  **Configurar el Despliegue:**
    *   Netlify detectará automáticamente que es un sitio estático. **No necesitas cambiar ninguna configuración de compilación.**
    *   El "Build command" debe estar vacío.
    *   El "Publish directory" debe estar vacío (o configurado en la raíz del proyecto).
    *   Haz clic en el botón **"Deploy site"**.

¡Y eso es todo! En menos de un minuto, Netlify desplegará tu sitio y te dará una URL pública (algo como `random-name.netlify.app`). Puedes configurar un dominio personalizado más tarde en la configuración del sitio de Netlify si lo deseas.
